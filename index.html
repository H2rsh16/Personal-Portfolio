Remote Code Execution System

Technologies Used -

MERN Full Stack App (MongoDB, React, Node, ExpressJS)

Created a Remote Code Execution system using the MERN stack, which includes MongoDB, Express.js, React.js, and Node.js. The system allows users to register by creating accounts with hashed passwords stored in MongoDB. Once registered, users can log in, receiving a JWT for session management. They can then write and submit code through a front-end editor. The back end features API endpoints for registration, login, and code execution, with Express handling the routing. Submitted code is executed in a sandboxed environment, such as Docker, to ensure security. The system implements robust security measures, including input validation, resource limits, and logging for auditing. Overall, the user flow involves registration, login, code submission, and safe retrieval of execution output.
